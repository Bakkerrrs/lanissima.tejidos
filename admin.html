Conectar con GitHub

Para publicar contenido necesitas un token personal de GitHub (se guarda solo en este navegador).

Cómo crear el token (una sola vez):
  1. Entra a github.com/settings/personal-access-tokens/new
  2. Nombre: «Lanissima admin». En Repository access elige Only select repositories → este repositorio.
  3. En Permissions → Repository permissions → Contents elige Read and write.
  4. Genera el token, cópialo y pégalo aquí.
